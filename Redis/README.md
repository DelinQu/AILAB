## 安装

- 直接在服务器宝塔面板里安装

![image-20210309124532557](https://i.loli.net/2021/03/09/ua1twLo9FrIERJQ.png)

- 也可以在apt源中安装

```zsh
sudo apt-get install redis-server
安装完成后，Redis服务器会自动启动。
使用ps -aux|grep redis命令可以看到服务器系统进程默认端口6379
```
